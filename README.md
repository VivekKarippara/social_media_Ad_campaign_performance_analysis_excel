# social_media_Ad_campaign_performance_analysis_excel
Excel-based data analytics project analyzing social media ad campaign performance for TrendWave. Includes KPI analysis (CPA, CPM, CTR, CCR), demographic insights, campaign optimization, ROI improvement strategies, and an interactive dashboard created for the Hack Studio September Edition Hackathon.

## 📌 Project Overview
This project was completed as part of the Hack Studio – September Edition Hackathon organized by Codingninjas 10x Club.
The objective was to analyze social media ad campaign performance data and generate actionable business insights regarding:
•	Audience targeting efficiency
•	Campaign effectiveness
•	Cost optimization
•	Conversion performance
•	ROI improvement strategies
The analysis was conducted using KPI-driven evaluation and interactive dashboard storytelling.

## 🎯 Business Problem
TrendWave invested heavily in paid social media campaigns across multiple audience segments but lacked clarity on:
•	Which demographics generate the best conversions
•	Which campaigns provide the highest ROI
•	Where ad spend is being wasted
•	How engagement varies across audiences
•	Which segments should receive future budget allocation
The project aims to build an analytical framework to optimize campaign performance and improve marketing decisions.

## ❓ Key Business Questions
1.	Which age and gender groups drive the highest conversions and ad spend?
2.	How effective are campaigns in converting impressions into clicks and clicks into sales?
3.	Which demographic segments have the highest and lowest CPA and CPM?
4.	Which campaigns should receive increased or reduced budget allocation?
5.	What trends can improve future advertising strategy?

## 📊 KPIs Used
The following performance metrics were calculated and analyzed:
KPI	Description:
CPM	Cost Per Mille
CPC	Cost Per Click
CTR	Click Through Rate
CPA	Cost Per Acquisition
CCR	Click Conversion Rate
KPI Formulas
CPM = (Spent / Impressions) × 1000

CPC = Spent / Clicks

CTR = (Clicks / Impressions) × 100

CPA = Spent / Approved Conversion

CCR = (Approved Conversion / Clicks) × 100

## 📈 Key Insights   
🔹 Best Performing Audience  
     •	30–34 Male Segment  
     •	Lowest CPA  
     •  Highest conversion efficiency  
     •	Most cost-effective audience  
🔹 Highest Engagement Audience  
     •	45–49 Female Segment  
     •	Highest CTR  
     •	Strong engagement   
     •	Poor conversion efficiency  
🔹 Campaign Performance  
  •	Campaign 916  
     •	Highest CTR  
     •	Lowest CPA  
     •	Best overall campaign  
  •	Campaign 1178  
     •	Lowest efficiency  
     •	Highest acquisition cost  
🔹 Cost Trends  
     •	CPA and CPM increase significantly with age  
     •	Younger audiences provide better ROI  
     •	Male-targeted campaigns generally perform better in conversions  

## 📌 Recommendations
•	Increase budget allocation for high-performing campaigns and demographics
•	Reduce spend on high-CPA audience groups
•	Optimize creatives for costly demographics
•	Improve post-click conversion experience
•	Continuously monitor CTR, CPA, CPM, and conversion trends
